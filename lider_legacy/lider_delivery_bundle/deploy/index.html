근거 중심 응답

검증된 사실, 추천 문장, 불확실성을 시각적으로 분리했습니다. Search 결과는 source chip 없이 단정하지 않도록 설계했습니다.

문서 추출 분리

문서 업로드는 추출 JSON → validator 경고 → 사람용 요약 순서로 분리했습니다. 추출 실패는 warning으로 드러납니다.

위험 액션 안전화

승인·변경·종료는 Preview 중심 UX로 변경했습니다. 권한과 missing checks가 먼저 보이며 execute는 비활성입니다.

Design fixes

가정 기반으로 바로 잡은 최종 디자인 오류

  • 긴 JSON / 긴 텍스트 overflow 방지
  • 모바일에서 카드가 두 줄 이상으로 자연스럽게 내려가도록 보정
  • CTA 위계 재정렬: Preview와 Execute 명확 분리
  • 탭 overflow 및 sticky header 개선
  • 상태 배지 대비와 키보드 focus ring 추가
